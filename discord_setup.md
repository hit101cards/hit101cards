# Hit101 Discord サーバー構築ガイド

## 基本設定

| 項目 | 設定値 |
|---|---|
| サーバー名 | `Hit101 Official` |
| サーバーアイコン | `client/public/apple-touch-icon.png` を流用 |
| サーバーバナー | `client/public/og-image.png` を流用（要Boost Lv1） |
| デフォルトチャンネル | `#welcome` |
| 通知設定 | `@mentions only`（メンバー流入時の通知抑制） |
| 認証レベル | `中` (Discordアカウント作成5分以上) |

---

## チャンネル構成

各カテゴリの下にチャンネルを配置する。チャンネル名は半角小文字＋ハイフン推奨（モバイルでも崩れない）。

### 📢 INFORMATION（読み取り専用）

| チャンネル | 説明 |
|---|---|
| `#welcome` | 新規参加者向け案内、リンク集 |
| `#rules` | サーバールール |
| `#announcements` | 公式お知らせ、新機能告知 |
| `#updates` | 開発進捗・パッチノート |

### 💬 COMMUNITY（一般会話）

| チャンネル | 説明 |
|---|---|
| `#general-jp` | 日本語の雑談 |
| `#general-en` | English chat |
| `#introductions` | 自己紹介 |
| `#screenshots` | スクショ・名場面共有 |

### 🎮 GAMEPLAY

| チャンネル | 説明 |
|---|---|
| `#looking-for-game` | 対戦相手募集（部屋ID共有） |
| `#strategy` | 戦略・テクニック議論 |
| `#ranking-talk` | ランキング・自己ベスト共有 |

### 🛠️ FEEDBACK

| チャンネル | 説明 |
|---|---|
| `#bug-reports` | バグ報告（テンプレ運用推奨） |
| `#feature-requests` | 機能要望 |
| `#help` | プレイ方法の質問 |

### 🔒 STAFF（運営のみ可視）

| チャンネル | 説明 |
|---|---|
| `#mod-chat` | モデレーター連絡 |
| `#mod-log` | Bot自動ログ（参加・脱退・削除等） |
| `#audit-log` | 不正検知ログ転記用 |

## ロール構成

ロールは**上から優先度順**にDiscord上で並べる。

| ロール名 | 色 | 役割 |
|---|---|---|
| `運営 (Owner)` | 金 #FFD700 | 全権限 |
| `モデレーター (Mod)` | 赤 #DC2626 | メンバー管理、メッセージ削除、Kick |
| `ベータテスター (Beta)` | 紫 #9333EA | 公開前テスト参加者、限定チャンネル可視 |
| `開発者と対戦済 (Played with Dev)` | 緑 #16A34A | 運営とプレイした記念ロール、雑談用 |
| `日本語 (JP)` | 青 #3B82F6 | 日本語チャンネル可視（自己取得） |
| `English (EN)` | 水 #06B6D4 | 英語チャンネル可視（自己取得） |
| `@everyone` | デフォ | 一般メンバー |

### 自己ロール取得（#roles チャンネル + リアクションBot）

```
🇯🇵 を押すと「日本語」ロール取得（#general-jp が見える）
🇬🇧 を押すと「English」ロール取得（#general-en が見える）
🐛 を押すと「ベータテスター募集」通知ロール取得
```

推奨Bot: **Carl-bot** または **Reaction Roles**（無料・公式）

### #rules
```
📜 **Hit101 サーバールール**

**1. 互いに敬意を持つ**
誹謗中傷・差別・嫌がらせ・煽り行為は禁止。

**2. チート禁止**
ゲーム内でのチート・自動化ツール・脆弱性悪用に関する助長・推奨は禁止。

**3. 健全なコンテンツ**
NSFW・グロ・違法・公序良俗に反するコンテンツの投稿禁止。

**4. スパム・宣伝抑制**
他サービスの大量宣伝・営業DMは禁止。自分のSNS紹介程度はOK。

**5. プライバシー**
本名・住所・電話番号など個人情報の自他共に投稿禁止。

**6. チャンネルの趣旨を守る**
雑談は #general-jp / #general-en へ。バグは #bug-reports へ。

**7. モデレーターの指示に従う**

違反時の対応:
⚠️ 警告 → ⏱️ タイムアウト → 🚫 BAN

質問は <#help> または DMで運営へ。
```


## 招待リンク戦略

1. **作成時**: 30日有効・無制限使用回数のリンクを発行
   - X / TikTok / Instagram のbioに貼付
2. **公開後**: ハンドル名Vanity URL `discord.gg/hit101cards` を取得（Boost Lv3要、後回しでOK）
3. **使い分け**:
   - SNS bio用: 1つ固定リンク
   - イベント告知用: 7日有効リンクを都度発行（流入経路分析用）
